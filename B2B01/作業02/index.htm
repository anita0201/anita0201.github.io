草莓

我們吃的草莓不是果實?!

「我們吃的草莓，其實不是它真正的果實」。果實是由子房發育，但草莓是花托膨脹後形成的「假果」，真正的果實是附在表皮上如芝麻般的小顆粒。此外，草莓雖然有個「莓」字，但它在植物學的定義上卻不屬於「莓果」，而是「多個果實聚集在花托上」的「聚合果」。

草莓種在地上& 高架栽培，差別在哪？

兩者最大差別在於病蟲害的管理。採用高架的方式種植草莓，能避免土壤傳播性危害的發生，但是必須以介質種植，因為缺少土壤礦物質的滋養，農友若沒有特別注意肥份的給予，在風味上則會略遜於土耕草莓。對農友而言，採收高架種植的草莓會比較方便，若有兼營觀光果園，消費者也比較喜歡。不過，考量的主要還是設施成本，所以目前產地主要還是以土耕為主。

草莓4好處及功效

1.預防大腦退化：草莓含有豐富的碘、維生素C與植化物，有助於維持神經系統正常運作、增加大腦的血流量及改善認知功能。而花青素也有助於防止腦神經老化，改善認知能力並預防大腦退化。
2. 預防癌症：草莓含有豐富的鞣花酸，它是一種天然酚類抗氧化劑，具有高度抗氧化能力，可修復體內受損的細胞，也可以抑制癌細胞擴散。而黃酮類化合物及原花青素，也具有強力的防癌作用。
3. 保護心血管：草莓含有花青素、兒茶素、槲皮素、山奈酚、鞣花酸等物質，這些物質可抑制人體體內的發炎反應、降低血壓，減少壞膽固醇的形成，達到降低罹患心血管疾病風險的功效。
4. 美白淡斑：草莓含豐富維生素 C ，可以促進皮膚新陳代謝，抑制皮膚酪胺酸脢活性，達到美白淡斑功效。
